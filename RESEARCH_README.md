# Wartungsplaner Handbook Summary

## Overview
Wartungsplaner is a software solution for managing maintenance, inspections and safety compliance of assets (machines, equipment, vehicles, etc.). It supports scheduling recurring inspection dates, tracking tasks, generating reports, and ensuring regulatory compliance (TRBS, BGV, BetrSichV, ISO 9001).

## Core Features
- Asset catalog with hierarchical tree (up to 4 levels), images, manuals, attachments.
- Creation of inspection objects and customizable checklists.
- Assignment of responsibilities (examiner, safety officer, etc.).
- Report generation with barcodes, PDF/Word export, templates.
- Calendar views (weekly, monthly, yearly) with Outlook sync and email reminders.
- Dashboard showing upcoming tasks, overdue items, KPI charts (pie/bar), utilization board.
- Controlling: maintenance KPIs, MTTR, MTBF, IH‑Quote.
- Automatic reminder center.
- Best‑practice start guide.
- Installation on local PC or network, multi‑user licensing per Windows login.
- Full program UI with quick‑access toolbar, zoom, symbol bar.
- Search, filter, and object tree navigation.
- Status ampel (red/yellow/green) for due dates.
- Detailed object view with tabs: reports, operational data, memos, attachments, links.
- Report list with status icons (open/closed, attachment indicator).
- Print options: PDF, Word, Excel, image export, combined PDF‑package.
- Batch processing and import/export via CSV/Excel/Access.
- User management with role‑based permissions (admin, standard). 
- Digital signature for report finalisation.
- Interval reports (auto‑generate recurring tasks).
- Resource & spare‑part management linked to reports.
- Hazard assessment module with risk matrix, groups, factors, scoring.
- Messaging/notification via email or messenger on status changes.
- Integration modules: RFID scanner, barcode scanner, mobile app (iOS/Android), Outlook sync, inventory import, custom extensions.
- Multi‑language support and customizable UI strings.
- Reporting engine with customizable layouts, logos, barcodes, conditional images.
- Export of dashboards to PDF/HTML for infomonitor displays.
- Data archiving and restoration.
- Audit trail logging changes per object/report.
- Advanced search & replace assistant for bulk edits.
- Online news feed and help system.

## Modules / Extensions
- Messenger/E‑mail notifications on status change.
- Hazard substance registry.
- Custom language packs.
- Scanner modules (barcode, RFID).
- Mobile app interface for data capture.
- Electroschnittstelle for Benning ST 760.
- Additional custom modules via executable plugins.

## Workflow Highlights
1. Define assets → create objects.
2. Set up inspection checklists and intervals.
3. Schedule tasks – appear in calendar/dashboard.
4. Perform inspections, record results, attach files.
5. Generate compliance reports, export to required formats.
6. Review KPIs, address overdue items via reminders.
7. Archive old data or migrate between installations.

## Compliance & Legal
References German occupational safety regulations (ArbSchG, ASiG), technical rules (TRBS), ISO 9001/9004 requirements, and documentation obligations for inspections and maintenance.
